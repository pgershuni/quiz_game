тесты:
    создание тестов:
        /api/tests, метод-post
        структура запроса:
             {'title': название теста,
              'about': о чем тест,
              'category': категория,
              'questions': [
                {'type': тип вороса (ord - простой/ rad - с вариантом выбора/ check - чекбоксы),
                 'question':
                    если ord:
                        сам вопрос
                    если rad или check:
                        [сам вопрос[варианты выбора]],
                 'answer':
                    если ord или rad:
                        ответ
                    если check:
                        [ответы]]},
              'is_private': приватность теста (True/False),
              'user_id': id пользователя}
        возвращает ключ
    получение всех тестов:
        /api/tests, метод-get
        возвращает список с тестами
        {tests: [тест1, тест2, тест3]}
    получение теста по ключу:
        /api/tests/ключ, метод-get
        возвращает тест
        {'test': {'about': 'о чем тест',
            'category': 'категория',
            'is_private': False,
            'key': 'ключ',
            'name': 'название теста',
            'questions': [
                    {'answer': 'ответ', 'question': 'опрос', 'type': 'ord'},
                    {'answer': 'ответ', 'question': ['сам вопрос', ['варианты выбора', 'варианты выбора1', 'варианты выбора2']], 'type': 'rad'},
                    {'answer': ['ответы', 'ответы1'], 'question': ['сам вопрос', ['варианты выбора', 'варианты выбора1', 'варианты выбора2']], 'type': 'check'}], 'user_id': 1}}

    удаление теста по ключу:
        /api/tests/ключ, метод-delete
        ничего не возвращает


пользователи:
    создание пользователей:
        /api/users, метод-post
        структура запроса:
             {'name': имя пользователя,
              'about': о пользователе,
              'login': логин,
              'password': пароль
             }
        возвращает id
    получение всех пользователей:
        /api/users, метод-get
        возвращает список с пользователями
    получение пользователя по id:
        /api/users/id, метод-get
        возвращает пользователя
    удаление пользователя по id:
        /api/users/id, метод-delete
        ничего не возвращает


ключи для подключения телеграм:
    ключ действует 10 минут

    создание ключа:
        /api/telegram_keys, метод-post
        структура запроса:
             {'user_id': id пользователя для которого предназначается ключ}
        возвращает ключ
    получение id пользователя по ключу:
        /api/telegram_keys/ключ, метод-get
        возвращает id пользователя
    удаление ключа по ключу:
        /api/telegram_keys/ключ, метод-delete
        ничего не возвращает


добавление пройденного теста юзеру:
    /api/passed_tests/id юзера, метод-get
    ничего не возвращает
просмотр категорий:
    /api/categories, метод-get
    пример результата:
        {'categories': [{'text': 'Химия'}, {'text': 'Физика'}, {'text': 'География'}]}
